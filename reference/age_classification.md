# Age classification

Example data set to create age classification with aggreations form 10
and 25 years classes.

## Usage

``` r
age_classification
```

## Format

A data frame:

- valuecode:

  Value code

- valuetext:

  Value text

- 10-years_classes:

  Aggregtation into 10 years classes

- 25-years_classes:

  Aggregtation into 25 years classes
