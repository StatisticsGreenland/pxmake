# Guess encoding of file

Guess encoding of file

## Usage

``` r
guess_file_encoding(path, prefer = list(`UTF-8` = 0.2, `ISO-8859-1` = 0.2))
```

## Arguments

- path:

  Path to file

- prefer:

  Named list of encodings, used to give higher confidence to specific
  encodings.

## Value

Character
