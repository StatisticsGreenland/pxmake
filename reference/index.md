# Package index

## Main function

- [`px()`](https://statisticsgreenland.github.io/pxmake/reference/px.md)
  : Create a px object

## Create files

- [`px_save()`](https://statisticsgreenland.github.io/pxmake/reference/px_save.md)
  : Save px object to file
- [`px_save_classification()`](https://statisticsgreenland.github.io/pxmake/reference/px_save_classification.md)
  : Save classification as .vs and .agg files

## Modify px object

Once a px object has been created, with
[`px()`](https://statisticsgreenland.github.io/pxmake/reference/px.md),
it can be modified with the functions below
