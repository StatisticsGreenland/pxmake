# Zips list

Combine two list, by zipping them together in the order `v1[1]`,
`v2[1]`, `v1[2]`, `v2[2]`, `v1[3]`, ...

## Usage

``` r
zip_vectors(v1, v2)
```

## Arguments

- v1:

  First vector Elements ends up on uneven indexes.

- v2:

  Second vector. Elements ends up on even indexes.

## Value

List
